# aleksandrmovsisian.github.io

Personal academic website of Aleksandr Movsisian — pure HTML/CSS/JS,
no frameworks, no build step. Deployed via GitHub Pages.

**See [PROJECT.md](PROJECT.md) for the full project map** — folder
structure, navigation scheme, design tokens, and change log.

## Run locally

```bash
python -m http.server 8000
# → http://localhost:8000
```

## Deploy

```bash
git add <files>
git commit -m "message"
git push
# GitHub Pages rebuilds automatically in ~1-2 minutes
```
